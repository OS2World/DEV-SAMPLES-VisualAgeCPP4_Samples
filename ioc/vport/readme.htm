Viewport Sample

Description

The View Port Sample illustrates the viewport's ability to scroll the client window that is not completely visible due to its size. This viewport scrolls a bitmap that is too large for the client area.

Concept

Sample program demonstrates IViewPort and IBitmapControl.

Task

Using the vertical and horizontal scroll bars, view the bitmap by moving a line or page at a time. Also, resize the frame window for different views.
