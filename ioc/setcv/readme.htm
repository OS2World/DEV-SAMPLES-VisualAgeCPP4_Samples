Set Canvas Sample

Description

The Set Canvas Sample demonstrates alignment of Open Class objects on vertical and horizontal set canvases. In addition, horizontal and vertical split canvases, and tab movement between different groups.

Concept

Sample program demonstrates ISetCanvas and ISplitCanvas.

Task

The three left canvases have vertical orientation and the right canvases have horizontal orientation. Each canvas has a different number of controls to demonstrate the arrangement differences.
