Tool Bar Button Sample

Description

The Tool Bar Button Sample demonstrates the OpenClass supplied toolbar buttons, flyover text, and information area text. Clicking a toolbar button displays the OpenClass ID. The tool bar buttons appear as bitmaps, bitmaps and text, or just text. The toolbar itself can be positioned on the top, bottom, left, right, or as a free floating window.

Concept

Sample program demonstrates IToolBar, IToolBarButton, IFlyText, IFlyOverHelpHandler.

Task

Change the toolbar views and locations.
