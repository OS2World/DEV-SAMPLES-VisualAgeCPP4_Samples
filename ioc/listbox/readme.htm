Listbox Sample

Description

The List Box Sample demonstrates adding and retrieving data from a listbox. You can add items, retrieve select items, or retrieve all the items to the second listbox. In addition, you can select or deselect all listbox items and test for exception handling.

Concept

Sample program demonstrates a listbox and accelerators.

Task

Select food items in the left listbox. Selecting the Read menu item places the selected items in the right listbox.
