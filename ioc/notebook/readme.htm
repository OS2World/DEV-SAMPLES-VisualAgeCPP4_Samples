Notebook Sample

Description

The notebook sample demonstrates the use of INotebook class by placing four bitmap controls on separate notebook pages. The notebook page tabs show a miniature version of each bitmap while each page shows the full size image. Because the bitmaps are placed in IViewport objects, you can scroll both horizontically and vertically whenever the bitmap does not fit. Color palette support is also demonstrated.

Concept

Sample program demonstrates a notebook, viewports, and bitmaps.

Task

Each notebook page has a viewport and a unique bitmap. Move to each notebook page using the notebook tabs and notebook scroll buttons.
