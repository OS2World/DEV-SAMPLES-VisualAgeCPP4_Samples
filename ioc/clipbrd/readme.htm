Clipboard Sample

Description

The Clipboard Sample illustrates a container with clipboard support. A popup menu appears over an object with cut, copy, and paste actions. Classes used include IContainerControl, IContainerObject, and IClipboard.

Concept

Create and display a container. Clipboard actions are cut, copy, and paste.

Task

The container has two objects. You can cut, copy, and paste these objects to the container.
