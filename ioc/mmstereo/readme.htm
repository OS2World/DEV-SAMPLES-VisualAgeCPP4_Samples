Multimedia Stereo Sample

Description

The MMSTEREO sample also uses many of the multimedia functions. It also illustrates the visual control panels provided as part of the multimedia classes. Visually it resembles a stereo rack system with amplifier, mixer, and video and CD players.

Concept

Sample program demonstrates amplifiers, mixers, and multimedia players.

Task

Choose a multimedia type from the three player choices. Except for CD, press the Load button to chose a wave or video file. Press the play, rewind, forward, and pause buttons. Adjust the volume and balance circular sliders.

Additonal Hardware and Software Requirements:

  • CD-ROM with a music CD installed
  • Supported sound card
