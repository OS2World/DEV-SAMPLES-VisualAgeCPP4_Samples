Direct Manipulation Sample 1

Description

The Direct Manipulation Sample 1 demonstrates dragging and dropping text from entryfields. By typing data into one entryfield, you can drag the text and drop into the other entryfield. Classes used include IDMEFItem, and IDMHandler.

Concept

Sample program demonstrates direct manipulation between entryfields.

Task

Highlight text in the source entryfield and drag to the target entryfield.
