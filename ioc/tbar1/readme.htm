Tool Bar Sample 1

Description

The Tool Bar Sample 1 demonstrates the toolbar, toolbar buttons, and flyover text within a simple editor. The tool bar buttons appear as bitmaps, bitmaps and text, or just text. The toolbar itself can be positioned on the top, bottom, left, right, free floating window, or hidden.

Concept

Sample program demonstrates IToolBar, IToolBarButton, IFlyText, IFlyOverHelpHandler, and IMultiLineEdit.

Task

The toolbar manipulates the multiline entryfield. The toolbars have bitmap, text, and bitmap with text views.

Special Notes:

The sample must run from the sample directory to load a file into the multiline entryfield.
