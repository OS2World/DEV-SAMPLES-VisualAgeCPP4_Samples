2DDocs Graphics Sample

Description

This sample contains code snippets that are used by the 2D Graphics documentation set. EXAMPLES.CPP contains all of the code samples themselves; the other files are support files for the application.

Concept

2DDocs demonstrates programming with the 2D Graphics classes. It is designed to complement the 2DCombo graphics sample program.

Task

From the menu, select the particular example you want to see. The Wagon example is interactive: click on it to select particular pieces of the graphic groups which make up the wagons.
