2DPaint Graphics Sample

Description

2DPaint is a version of the 2DDraw sample written using MGraphic objects. It has a toolbar and menu items for drawing the various MGraphic primitives on a canvas.

Concept

The 2DPaint sample demonstrates programming with the 2D Graphics classes.

Task

Use like the 2DDraw sample.

Special Notes

On Windows 95, there are some differences from Windows NT.  In particular, text and images do not support rotation or perform other transforms. Also, text does not scale, but images will to some extent. All operations work correctly on NT.
