Multimedia Remote Control Sample

Description

The MMREMOTE sample illustrates many of the multimedia functions. It takes the form of the universal remote control, which controls the CD player, video player, midi player, and wave player devices.

Concept

Sample program demonstrates multicell canvases, flyover help, animated pushbuttons, and multimedia controls for CD, video, midi, and wave.

Task

Select a multimedia type from the four choices. Except for CD, press the Load button to chose a wave, midi, or video file. Press the play, rewind, forward, and pause buttons.

Additonal Hardware and Software Requirements:

  • CD-ROM with a music CD installed
  • Supported sound card
