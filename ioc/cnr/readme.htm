Container Sample

Description

The Container Sample illustrates a container and container objects. The container has parent container objects and child container objects. The initial view is treeIconView but can be changed to any view including textView, flowedTextView, nameView, detailsView, treeTextView, treeNameView, and iconView. The sample can also select or deselect all container objects. In addition, a filter is provided for viewing only a select parent container object. Classes used include IContainerControl, IContainerObject, IContainerColumn, and IContainerControl::FilterFn.

Concept

Creates and displays a container. The menu items change container views, select or deselect container objects (in non-tree views), and filters to display only selected items ( PM compatible container only)

Task

From the menu, you can change the view for the container and sort or filter the container's contents.
